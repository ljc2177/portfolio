Visualizing
William Still
and the
Underground
Railroad

November 2022.
Output: Scrollable narrative website with
corresponding interactive map.

This project draws upon the data William Still collected as he assisted freedom-seekers along the Underground Railroad, which was recently digitized by Nick Sacco. In visualizing this data, we can not only discover trends and patterns, but we can fully recognize Still's invaluable work. Completed as part of my work in the MS Computational Design Practices program at Columbia University.

Tools & Techniques Used: HTML, CSS, JS, D3, GeoJson
